## 🎯 2. Conceito: Utility First

> Classe que faz **uma única coisa**

.grid {  
  display: grid;  
}  
  
.grid-flow-col {  
  grid-auto-flow: column;  
}

✔ Você aplica onde quiser  
✔ Reutilizável  
✔ Muito usado em frameworks modernos (ex: Tailwind)

# 🎨 8. Card com Degradê (Overlay)

## Estrutura

.card {  
  position: relative;  
  overflow: hidden;  
  border-radius: 4px;  
}

---

## Pseudo-elemento (::before)

.card::before {  
  content: "";  
  position: absolute;  
  inset: 0;  
  
  background: linear-gradient(  
    180deg,  
    transparent 0%,  
    rgba(0, 0, 0, 0.8) 100%  
  );  
}

---

## 🧠 Conceitos importantes

### 🔹 `position: relative`

Define referência pro filho absoluto

### 🔹 `position: absolute`

Permite posicionamento livre

### 🔹 `inset: 0`

Equivale a:

top: 0;  
right: 0;  
bottom: 0;  
left: 0;

---

## 🎯 Linear Gradient

background: linear-gradient(  
  180deg,  
  transparent 0%,  
  black 100%  
);

✔ 180deg → de cima pra baixo  
✔ Começa transparente  
✔ Termina escuro

---

# ⚠️ Problemas comuns (e solução)

### ❌ Scroll horizontal

✔ Causa: imagem maior que container  
✔ Solução:

img {  
  max-width: 100%;  
}

---

### ❌ Imagem esticando

✔ Causa: height sem ajuste  
✔ Solução:

object-fit: cover;

---

### ❌ Borda não funcionando

✔ Causa: overflow da imagem  
✔ Solução:

overflow: hidden;

---

# 🧾 Resumo rápido (prova)

- Utility First = classe faz 1 coisa
- `grid-auto-flow: column` = elementos em linha
- `margin-inline: auto` = centraliza horizontal
- `object-fit: cover` = mantém proporção da imagem
- `overflow: hidden` = corta excesso
- `::before` + `linear-gradient` = overlay com degradê
- `inset: 0` = ocupa tudo
